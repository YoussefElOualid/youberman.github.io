Installation Requirement
<script href="jquery.js"><script>
<script href="jsonEngDesk.js"><script>
Attr JSONEngDesk
First Attr Description
json-url Url Or String Object
json-output Print Keys In Object JSON
json-click Url Or String Object
tag Create Elemenet Tag
tag-class Add Class To tag
tag-parent Create Elemenet Tag in ELement Parent
tag-parent-class Add Class To tag parent
return-output Return html to html element

Example 1

Example
Source html
<ul json-url='[{"nom":"Example 1"},{"nom":"Example 2"},{"nom":"Example 3"},{"nom":"Example 4"}]' json-output="nom" tag="li" > <ul>

Example 2

Fichier JSON
Source html
<div json-url="json.php" json-output="id,Prenom,nom" tag="label" tag-class="col-md-4 col-xs-4 text-primary" tag-parent-class="form-group" tag-parent="div"> <div>

Example 3

EXtern URL
Source html
